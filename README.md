# Evergreen Prompt

Evergreen prompt is an interactive CLI build using [Go Prompt](https://github.com/c-bata/go-prompt) and the evergreen cli.

## installation

Build from source
```
go build

# start the cli
./evergreen-prompt 
```

Or install with brew.

```bash
brew tap chatton/homebrew-tap
brew install evergreen-prompt

# start the cli
evergreen-prompt
```
